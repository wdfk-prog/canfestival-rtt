UNS16  index,
UNS8  subIndex,
UNS32  abortCode 
)

Called when an internal SDO abort occurs.

Release the line * Only if server * If client, the line must be released manually in the core application. The reason of that is to permit the program to read the transfers structure before its reset, because many informations are stored on it : index, subindex, data received or trasmited, ... In all cases, sends a SDO abort.
