void master402_post_emcy ( CO_Data *  d,
UNS8  nodeID,
UNS16  errCode,
UNS8  errReg,
const UNS8  errSpec[5] 
) 
@brief  主站节点emcy紧急事件回调.
@param  None.
@retval None.
@note   None.

errReg:如果某一特定的错误发生相应位置 1b。通用错误位是强制设备必须支持的，其它可选。任何错误产生都将置位通用错误。 位 M/O 意义 0 M 通用错误 1 O 电流 2 O 电压 3 O 温度 4 O 通信错误(溢出、错误状态) 5 O 设备协议指定 6 O 保留(始终为 0b) 7 O 制造商指定

ErrorCode: 6200 用户软件 -常规错误[] ErrorCode: 3130 主回路电源异常[驱动器掉电] ErrorCode: 8200 协议错误–常规 AL121 当上位机发送 PDO 给驱动器时，PDO 所指定的对象字典 Index 号码不正确，导致驱动器无法辨识。 AL124 PDO所欲存取的对象字典范围错误 讯息中的数据超出指定对象字典的范围[例如写入速度4000超出0~3000范围] ErrorCode: 8110 CAN overflow (object loss) AL113 TxPDO 传送失败
