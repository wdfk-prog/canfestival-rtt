详细描述

The heartbeat mechanism for a device is established through cyclically transmitting a message by a heartbeat producer.

One or more devices in the network are aware of this heartbeat message. If the heartbeat cycle fails for the heartbeat producer the local application on the heartbeat consumer will be informed about that event.
